### EXERCISE 0 

---

Create a new empty Sway project

```bash
$ forc init hello_world
$ cd hello_world
$ tree .
.
├── Cargo.toml
├── Forc.toml
├── src
│   └── main.sw
└── tests
    └── harness.rs
```

---

Inspect `src/main.sw`

```rust
script;

fn main() {

}
```

---

Inspect `Forc.toml`

```toml
[project]
authors = ["Mohammad"]
entry = "main.sw"
license = "Apache-2.0"
name = "hello_world"

[dependencies]
std = { git = "https://github.com/FuelLabs/sway-lib-std" }
```

---

Build the project

```bash
$ forc build
  ...
  Compiled library "std".
  Compiled script "hello_world".
  Bytecode size is 28 bytes.
```

---

Start a local Fuel full node

```bash
$ fuel-core --db-type=in-memory
```
